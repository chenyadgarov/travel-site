About The Website

אתר שנבנה כדי לשמור חוויות

האתר התחיל מהרצון לתעד את ירח הדבש שלנו, אבל נבנה מראש כאתר מסעות כללי. הרעיון הוא ליצור מקום אחד שמרכז זיכרונות, תמונות והמלצות מטיולים שונים.

מבחינה לימודית, הפרויקט עזר לי לתרגל בניית אתר מרובה עמודים, עיצוב רספונסיבי, עבודה עם תמונות, ניווט משותף ו־JavaScript בסיסי.

עמודי טיולים

כל טיול יכול לקבל עמוד משלו עם סיפור, יעדים, תמונות והמלצות. כך האתר לא מוגבל רק לטיול אחד.

יעדים

עמודים לפי מדינות וערים, כדי לארגן את התוכן בצורה ברורה.

חוויות

מקום לרגעים מיוחדים, מקומות שאהבנו וטיפים אישיים מהדרך.

גלריה

אזור תמונות שיכול להתפתח בהמשך לגלריה עם סינון לפי טיול או יעד.

Technical Side

מה הפרויקט מציג מבחינה טכנית?

HTML

מבנה עמודים

שימוש ב־header, main, section, article ו־footer ליצירת מבנה ברור.

CSS

עיצוב רספונסיבי

שימוש ב־Flex, ריווחים, Border, Shadow, Hover ו־Media Queries.

JavaScript

ניווט ואינטראקציות

טעינת תפריט משותף ועבודה עם DOM להצגת תוכן לפי בחירת המשתמש.
